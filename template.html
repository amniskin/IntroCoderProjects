Let's make some boxes!

Written by: Aaron Niskin

In this project, we will study CSS by making some boxes with certain properties. To see what the final version should look like, check out the finished product.

Prerequisites:

  • A computer to use for a bit.
  • Desire to learn web development
  • Basic familiarity with html

Learning Objectives:

  • Styling elements via CSS
  • Learn a few CSS properties

Step 1: an HTML file

You should already know how to do this, so this step should be very easy. If you are unsure of how to go about this, I suggest you try out the HTML basics code project.

<!DOCTYPE html>

<html>
  <head>
  </head>
  <body>
  </body>
</html>

Step 2: Let's make a CSS file

Firstly, we are going to want to create a new file called style.css (it doesn't have to be named style, but that's a good convention, so let's stick with it). Inside this file is where we will put all of our CSS rules. The problem is, how do we tell the HTML file that we're going to be using a style sheet? For this, we have the link tag.

In the head of your html file, let's go ahead and put a link tag with three attributes. The first attribute is called type and is used to specify the type of file we're linking. In order to tell our document that we're linking a text css file, we set the value to "text/css". The second attribute is called rel and we set that value to "stylesheet". The third value tells the document where it can find the stylesheet we're linking and is called href with the value being the path to the CSS file. Go ahead and link your CSS file. In the next step, we'll make sure your link is working correctly.

<html>
  <head>
    <link type='text/css' rel='stylesheet' href='/css/lecture2.css'>
  </head>
  <body>
  </body>
</html>

Step 3: Let's put some headers on it

Since the background color extends over the entire body (regarless of the size of the screen, etc), we want to set the background color of our body. To set the backroung property of an HTML tag, we use this syntax:

tag {
  property1: value1;
  property2: tag2;
  ...
}

So, to set the background-color property of our body, our code should look like:

body {
  background-color: color-choice;
}

In CSS colors can be specified via their color name (if CSS knows about this color), their unicode index, their rgb (red/green/blue) value, or even their rgba (where A denotes the opacity with 1 being fully opaque and 0 being fully transparent). In our case, we want the rgb value to be 40, 40, 40. To do this, we can set the background-color property to rgb(40, 40, 40). Go ahead and do that now.

body {
  background-color: rgb(40, 40, 40);
}
